## 框架概述 · Overview

**中文**

AIGF（AI Governance Framework）是一套结构化的企业 AI 治理方法论。核心命题是：**在 AI 快速发展的不确定性中建立确定性锚点，将组织改造成"AI 战略执行机器"。**

框架采用 **4 层级 × 5 维度** 矩阵结构，生成 20 个独立但相互关联的执行模块。每个模块回答同一组核心问题：*为什么做、做什么、怎么做*，并配有路线图、评估工具、关键指标与检查清单，可直接用于 AI 治理委员会的议题决策。

**English**

AIGF is a structured methodology for enterprise AI governance. Its core premise: **establish certainty anchors amid the uncertainty of rapid AI development, and transform the organization into an "AI strategy execution machine."**

The framework uses a **4-layer × 5-dimension** matrix to produce 20 independent but interrelated execution modules. Each module answers the same set of questions — *why, what, and how* — and ships with roadmaps, assessment tools, KPIs, and checklists ready for direct use in AI governance committee agendas.

每个单元格的标识由**维度字母（列）+ 层级字母（行）**组成，例如 `CG` = 变革维度 × 治理层。

### 部署到 GitHub Pages · Deploy to GitHub Pages

```bash
# 克隆仓库 / Clone the repo
git clone https://github.com/your-username/aigf.git
cd aigf

# 确认所有 21 个 HTML 文件都在根目录 / Verify all 21 HTML files are in root
ls *.html | wc -l   # should output: 21

# 推送到 main 分支 / Push to main branch
git add .
git commit -m "init: AIGF v0.1"
git push origin main

# 在 GitHub 仓库设置中启用 GitHub Pages，Source 选择 main 分支根目录
# In GitHub repo Settings → Pages → Source: main branch / root
```

部署完成后，访问 `https://your-username.github.io/aigf/` 即可。
